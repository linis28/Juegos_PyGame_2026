# 🎮 Juegos PyGame 2026

Este proyecto contiene el desarrollo de una colección de juegos educativos y prácticos utilizando Python y la biblioteca PyGame.

El objetivo principal es fortalecer habilidades de programación mediante la creación de videojuegos simples, aplicando conceptos de lógica, estructuras de control, eventos y gráficos.
